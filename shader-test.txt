=== SHADER TEST SUITE USAGE ===

1. In-App Testing:
   - Navigate to "Shader Test" page in the app
   - Click "Run Tests" to test all shaders
   - Filter by category if needed
   - Click on failed tests to see errors and generated WGSL

2. CLI Testing (while app is running):
   npm run shader-test start              # Start tests
   npm run shader-test status             # Check status  
   npm run shader-test results            # Get results
   npm run shader-test watch              # Watch in real-time
   npm run shader-test error "Effects/Blur"  # Get error details
   
3. Copilot Workflow:
   - Run: node shader-test-cli.js watch
   - This connects to the app and shows test progress
   - Get detailed errors with: node shader-test-cli.js error "<shader-id>"
   - After fixing the ISF parser, run: node shader-test-cli.js reload
   - Re-run tests: node shader-test-cli.js start
